VitePress简介 ​

VitePress 是 VuePress 小兄弟, 基于 Vite构建。我们喜爱VuePress，但是它是基于 Webpack 构建。为了一个只有几个简单页面的简单文档站点启动开发服务器所需的时间正变得让人难以忍受。即使是HMR热更新也需要几秒钟的时间才能在浏览器中显示出来。

从 VuePress 改进的地方有以下几点：

  • 开发服务器启动更快
  • 热更新更快
  • 构建更快(内部使用 Rollup)

部署步骤 ​

  1. 新建项目文件夹，在项目文件夹运行 cmd运行npm init初始化package.json

  2. 继续运行npm i -D vitepress安装VitePress

  3. 在package.json中添加脚本

    "scripts": {
    "docs:dev": "vitepress dev docs",
    "docs:build": "vitepress build docs",
    "docs:serve": "vitepress serve docs"
  }

  4. 运行 npm run docs:dev

  1. 打开local，会看到404，因为并未找到主页

  3. 在 docs/.vitepress/路径下创建config.js配置文件，设置整个网站的配置

    js
    module.exports = {

    ],
    // 主题

            { text: '知识', link: '/knowledge/Css' },
            { text: '生活', link: '/life/travel' },
            { text: '记录', link: '/rec/vitepress' },

        sidebar: { // 侧边栏，可以分组
            "/knowledge/": [

                    text: "前端知识",

                            text: "Css",
                            link: "/knowledge/Css",
                        },
                        {
                            text: "Html",
                            link: "/knowledge/Html",
                        }
                    ],
                },
            ],

    主题配置 themeConfig对象中可配置头部右上导航栏（nav）和侧边导航栏（sidebar）

    nav为一个数组，里面每一个对象对应一个导航栏项目

    ​ - text：导航标题

    ​ - link：导航地址

    js
    { text: '知识', link: '/knowledge/Css' },
        { text: '生活', link: '/life/travel' },
        { text: '记录', link: '/rec/vitepress' },

    sidebar为一个对象，对象key为设置在哪个路径下显示侧边栏，value设置侧边栏标题和侧边栏项目

    js
    sidebar: { // 侧边栏，可以分组
